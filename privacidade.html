Freelas Now — Política de Privacidade

← Voltar ao site

Última atualização: 19 de abril de 2026

A Freelas Now ("nós", "nosso") respeita sua privacidade e está comprometida em proteger os dados pessoais que você compartilha conosco ao usar nosso aplicativo e serviços.

1. Quem somos

Freelas Now é uma plataforma que conecta freelancers e empresas para contratações de diárias. O responsável pelo tratamento de dados é Bruno Gercke, com contato disponível em brunogercke@gmail.com.

2. Dados que coletamos

Coletamos apenas os dados necessários para o funcionamento do serviço:

  • Dados de cadastro: nome completo, e-mail, telefone e senha (armazenada de forma criptografada).
  • Documentos: CPF (freelancers) e CNPJ (empresas), necessários para processamento de pagamentos e conformidade legal.
  • Localização: coordenadas GPS do dispositivo, coletadas apenas quando o freelancer está com status "Disponível" para facilitar buscas por proximidade. A coleta é interrompida ao mudar para "Offline".
  • Dados financeiros: histórico de pagamentos, saldo de carteira e chave PIX. Dados de cartão de crédito são processados diretamente pelo Mercado Pago e não são armazenados em nossos servidores.
  • Avaliações e contratos: registros de contratações, avaliações e comunicações relacionadas aos serviços prestados.
  • Dados técnicos: token de notificação push (FCM), endereço IP e logs de acesso para fins de segurança e diagnóstico de erros.
  • Dados de uso: informações anônimas sobre como você interage com o aplicativo, como telas visitadas e funcionalidades utilizadas. Esses dados são coletados de forma agregada pelo Google Firebase Analytics e não permitem sua identificação pessoal.

3. Como usamos os dados

  • Autenticação e gerenciamento de conta.
  • Exibição de freelancers próximos no mapa para empresas contratantes.
  • Processamento de pagamentos, saques e transferências via PIX.
  • Envio de notificações push sobre contratações, pagamentos e avisos do sistema.
  • Cálculo de métricas de desempenho (avaliação média, pontualidade, ranking).
  • Análise de uso do aplicativo para melhoria contínua da experiência do usuário, utilizando dados anônimos e agregados.
  • Prevenção de fraudes e conformidade com obrigações legais.
  • Resolução de disputas entre usuários, incluindo análise por inteligência artificial.

4. Compartilhamento de dados

Não vendemos seus dados. Compartilhamos informações apenas com os seguintes prestadores de serviço, estritamente necessários para o funcionamento da plataforma:

  • Google Firebase — autenticação, banco de dados, armazenamento de arquivos, notificações push e analytics de uso. Política de privacidade.
  • Mercado Pago — processamento de pagamentos e assinaturas premium. Política de privacidade.
  • Railway — hospedagem do servidor backend. Política de privacidade.
  • Anthropic (Claude) — análise de disputas por inteligência artificial. Os dados enviados são limitados ao contexto da disputa. Política de privacidade.

Podemos divulgar dados quando exigido por lei, ordem judicial ou autoridade competente.

5. Localização

O app solicita permissão de localização em segundo plano exclusivamente para freelancers que ativam o status "Disponível". A localização é usada para:

  • Exibir o freelancer no mapa para empresas próximas.
  • Calcular distância e tempo estimado de chegada.

A coleta de localização cessa imediatamente ao definir o status como "Offline" ou ao encerrar o aplicativo.

6. Retenção de dados

  • Dados de conta: mantidos enquanto a conta estiver ativa.
  • Histórico de pagamentos: retidos por 5 anos para fins fiscais e legais.
  • Logs técnicos: excluídos automaticamente após 90 dias.
  • Dados de localização: não são armazenados de forma permanente; apenas a última posição conhecida é mantida.

7. Seus direitos (LGPD)

Conforme a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você tem direito a:

  • Confirmar a existência de tratamento de seus dados.
  • Acessar os dados que temos sobre você.
  • Corrigir dados incompletos, inexatos ou desatualizados.
  • Solicitar a anonimização, bloqueio ou eliminação de dados desnecessários.
  • Revogar o consentimento a qualquer momento.
  • Solicitar a portabilidade dos seus dados.

Para exercer qualquer desses direitos, entre em contato: brunogercke@gmail.com.

8. Segurança

Adotamos as seguintes medidas para proteger seus dados:

  • Comunicação criptografada via HTTPS/TLS em todas as requisições.
  • Autenticação via Firebase com tokens JWT de curta duração.
  • Senhas nunca armazenadas em texto puro.
  • Monitoramento de erros via Firebase Crashlytics.
  • Credenciais de produção gerenciadas via variáveis de ambiente, nunca no código-fonte.

9. Menores de idade

O Freelas Now é destinado exclusivamente a maiores de 18 anos. Não coletamos intencionalmente dados de menores. Se identificarmos um cadastro de menor de idade, a conta será encerrada imediatamente.

10. Alterações nesta política

Podemos atualizar esta política periodicamente. Em caso de alterações relevantes, notificaremos os usuários pelo app com pelo menos 15 dias de antecedência. O uso continuado do serviço após a notificação implica concordância com a nova versão.
